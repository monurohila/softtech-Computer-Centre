# Softech Computer Centre - Next.js Demo

A fully functional responsive demo website for Softech Computer Centre, Ganaur.

## Pages included
- Home
- About Us
- Certification Courses
- Computer Courses
- Results with demo search
- Gallery
- Admissions with demo form
- Blog
- Contact with demo form
- Privacy Policy

## Demo result search
Try these values on the Results page:
- `SFT1001`
- `SFT1002`
- `REG2025002`

## Run locally
```bash
npm install
npm run dev
```
Open `http://localhost:3000`.

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Go to Vercel > Add New Project.
3. Import the GitHub repository.
4. Click Deploy.

## Notes
Forms are front-end demo forms with success messages. Connect them to email, Google Sheets, Supabase, Firebase, or a CRM for production.
