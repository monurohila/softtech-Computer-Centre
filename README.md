# Softech Computer Centre Ganaur - Node.js Website

A responsive SEO-friendly Node.js/Express website based on the approved homepage design.

## Run locally
```bash
npm install
npm start
```
Open: http://localhost:3000

## Deploy on Vercel
1. Upload this folder to GitHub.
2. Import the GitHub repo in Vercel.
3. Framework preset: Other.
4. Build command: `npm install`
5. Start command: `npm start`

## Pages
Home, About Us, Certification Courses, Computer Courses, Results, Gallery, Admissions, Blog, Contact Us.
