Softech Computer Centre Ganaur WordPress Theme

Installation:
1. Go to WordPress Admin > Appearance > Themes > Add New > Upload Theme.
2. Upload softech-wp-theme.zip and activate.
3. Create these pages: Home, About Us, Certification Courses, Computer Courses, Results, Gallery, Student Testimonials, Admissions, Blog / News, Contact Us.
4. Go to Settings > Reading and set Home as the static homepage.
5. Go to Appearance > Menus and add the 10 pages to Primary Menu.
6. Replace phone/email/address in header.php/footer.php or request a dynamic Customizer version.
7. Recommended plugins: Rank Math SEO, WPForms, LiteSpeed Cache, Smush, Wordfence, UpdraftPlus.

Included:
- Responsive homepage with hero slider
- Certification/course cards
- Results search UI placeholder
- Admission CTA
- WhatsApp floating button
- SEO-friendly title support and meta description
